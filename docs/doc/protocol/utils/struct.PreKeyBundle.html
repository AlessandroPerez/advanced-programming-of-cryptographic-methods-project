Expand description

A PreKeyBundle contains the public keys and signature published by a recipient, used by an initiator to establish a shared secret using the X3DH key agreement protocol.

Fields§

§verifying_key: VerifyingKey

The recipient’s identity signing key (Ed25519), used to verify sig. For more information, see VerifyingKey.

§ik: PublicKey

The recipient’s identity public key. For more information, see PublicKey.

§spk: PublicKey

The recipient’s signed public pre-key. For more information, see PublicKey.

§sig: Signature

A signature of the spk, signed by the identity signing key. For more information, see Signature.

§otpk: Vec<PublicKey>

One or more ephemeral one-time pre-keys, X25519 public keys. If present, the initiator may use one to enhance forward secrecy. For more information, see PublicKey.

pub(crate) const BASE_SIZE: usize = 160usize

The total byte size of the pre-key bundle, which includes three Curve25519 public keys and one signature. This constant is used to verify the expected size of a PreKeyBundle.
